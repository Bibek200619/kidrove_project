# Kidrove Workshop Page Skill

You are building a polished workshop landing page for a children’s education platform.

## Goal

Create a complete full-stack assignment project with:

- React.js frontend
- TypeScript
- Tailwind CSS
- Express.js backend
- Registration form
- POST /api/enquiry endpoint

## Design direction

The page should be friendly, bright, trustworthy, and suitable for parents browsing a children’s workshop.

Use visual ideas like:

- Soft gradients
- Rounded cards
- Playful icons
- Light background
- Colorful CTAs
- Clear workshop information
- Friendly illustrations or abstract shapes
- Smooth but subtle animations

## Required page sections

Build these sections in order:

1. Hero
2. Workshop Details
3. Learning Outcomes
4. FAQ
5. Registration Form
6. Footer

## Required workshop data

Use exactly:

- Workshop Name: AI & Robotics Summer Workshop
- Age Group: 8–14 Years
- Duration: 4 Weeks
- Mode: Online
- Fee: ₹2,999
- Start Date: 15 July 2026

## Learning outcomes

Include at least five:

- Understand the basics of Artificial Intelligence
- Learn how robots sense, think, and respond
- Build simple logic-based robotics projects
- Explore beginner-friendly coding concepts
- Develop problem-solving and creative thinking skills
- Understand real-world uses of AI and robotics
- Work on mini projects during the workshop

## FAQ requirements

Include at least three FAQs:

- Is this workshop suitable for beginners?
- Will the workshop be conducted online?
- Does my child need prior coding experience?
- Will students work on projects?

## Form behavior

The registration form must:

- Validate name, email, and phone
- Show validation errors
- Show loading state
- Submit to backend
- Show success response
- Show failure response

## Backend behavior

Create Express API:

POST /api/enquiry

Validate required fields and return JSON success/error responses.

MongoDB is optional. Do not add MongoDB unless asked.

## Completion checklist

Before finishing, verify:

- Page is responsive
- Required sections exist
- Form submits to backend
- Backend validates inputs
- Code is component-based
- README explains how to run frontend and backend
